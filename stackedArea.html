Norwegian Development Fund Disbursements 2000-2015

Which global region received the most funds between 2000-2015?

This chart shows the cumulative amount of funds each geographical regions has received from the Norwegian Fund between 2000-2015.

  • Africa             12,307,239
  • Asia                 6,093,064
  • America          3,400,477
  • Middle East    2,492,061
  • Europe            1,867,036
  • Oceania           18,119

Interactions

Select the type of Stacked Area Chart by clicking "Stacked", "Stream", or "Expanded".

Include or exclude regions by selecting the region buttons on the type right.

Hover over each year to reveal the amount of funds received per region.

About this page

This chart was built using nvd3 and angularjs.

User Interface Events

The user can also change the type of graph selecting either "Stacked", "Stream", or "Expanded". Regions can also be included or excluded through selection.

Annotations

The region name and funds received is displayed when the user hovers over each year point. The y-axis represents the total amount disbursed in (000ʻs usd). The x-axis represents each year.

Triggers

A display each regionʻs annual funds when the tooltip is hovered over the data. The user can also change the type of graph selecting either "Stacked", "Stream", or "Expanded". Regions can also be included or excluded through selection.

Parameters

The chart is based on the data provided in funds.csv. The parameters are "key"= Recipient Region, "values"=[Year, Disbursements (1000 $)].
